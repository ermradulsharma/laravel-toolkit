# Security Policy

## Supported Versions

The Skywalker Toolkit is maintained with a highly stable API. We support the major 1.x branch across a wide array of PHP and Laravel releases.

| Version | Supported          | Compatible PHP | Compatible Laravel Baseline      |
| ------- | ------------------ | -------------- | -------------------------------- |
| 1.x     | :white_check_mark: | 7.4 - 9.0      | 5.5, 6.20+, 7.30+, 8.83+, 9.52+, 10.48+, 11.33+, 12.0+ |

## Reporting a Vulnerability

Security forms the backbone of the Aether Security Suite embedded within this toolkit. If you discover any security related issues or cryptographic vulnerabilities, please email **sywalkerlknw@gmail.com** directly instead of using the public issue tracker.

We will acknowledge your report within 24 hours, perform a root-cause analysis, and deploy a secure patch simultaneously across legacy and modern branches.
